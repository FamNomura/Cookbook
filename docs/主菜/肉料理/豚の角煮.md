# 豚の角煮

とろとろに煮込んだ定番の角煮です。

## 材料
* 豚バラブロック: 500g
* 大根: 1/2本
* 生姜: 1片
* ゆで卵: 4個

## 調味料
* 水: 400ml
* 酒: 100ml
* 醤油: 大さじ4
* 砂糖: 大さじ3

## 手順
1. 豚肉を3cm幅に切る。
2. フライパンで表面を焼き、脂を落とす。
3. 鍋に肉、水、酒、生姜を入れ、弱火で1時間下茹でする。
4. 大根と調味料を加え、さらに30分煮込む。
5. 最後にゆで卵を入れ、冷ましながら味を染み込ませる。

## メモ
* 下茹での汁はスープに使えます。
* 圧力鍋を使う場合は加圧15分。
